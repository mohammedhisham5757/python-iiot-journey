## 🛠️ Projects Built

### 🔴 Project 1 — Smart Sensor Monitor
> Simulates 5 industrial sensors, checks thresholds, generates CRITICAL/WARNING/NORMAL alerts
- **Skills:** Variables, Lists, Dictionaries, Functions, Loops
- **Stack:** Pure Python

### 🟡 Project 2 — Industrial Data Logger
> Logs factory sensor readings with timestamps to JSON files, generates analysis reports
- **Skills:** File I/O, JSON, datetime, Functions, Error Handling
- **Stack:** Python, json, datetime, random

### 🟢 Project 3 — OOP Sensor Network
> Full object-oriented sensor network with base class + 3 child sensor types
- **Skills:** OOP, Inheritance, Packages, Docstrings
- **Stack:** Python OOP

### 🔵 Project 4 — Factory Data Analyzer
> Processes 500 sensor readings, generates charts, detects fault patterns
- **Skills:** NumPy, Pandas, Matplotlib
- **Stack:** Python, NumPy, Pandas, Matplotlib

### 🟣 Project 5 — Advanced Data Pipeline
> Complete pipeline: simulate → clean → analyze → visualize → log → thread
- **Skills:** Threading, Logging, Regex, Virtual Environments
- **Stack:** Python, pandas, matplotlib, logging, threading

### ⚫ Project 6 — Predictive Maintenance ML
> Detects motor anomalies using Isolation Forest before failure occurs
- **Skills:** scikit-learn, IsolationForest, time series, seaborn
- **Stack:** Python, scikit-learn, pandas, seaborn, matplotlib

### 🏆 Capstone — Industrial Monitoring System
> Complete IIoT monitoring system with OOP + ML + REST API + Database + Tests
- **Skills:** Full-stack Python, FastAPI, SQLite, pytest, clean code
- **Stack:** Python, FastAPI, SQLite, scikit-learn, pytest, black
